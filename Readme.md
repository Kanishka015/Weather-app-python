# Weather App using Tkinter and OpenWeatherMap API

## Project Description

This is a simple **Weather Forecasting Application** built using **Python** and **Tkinter**. It integrates with the **OpenWeatherMap API** to fetch real-time weather data for a city entered by the user. The app provides a clean and responsive GUI to display temperature, weather conditions, humidity, and pressure.

## Features

- **City-based Weather**: Enter the city name and fetch current weather data.
- **Real-time Data**: Uses the OpenWeatherMap API to provide accurate weather information.
- **GUI Interface**: Built with Tkinter for ease of use.

## Technologies Used

- **Python**: Main programming language for the project.
- **Tkinter**: GUI library for building the interface.
- **OpenWeatherMap API**: Provides weather data.
- **Requests**: Library to make HTTP requests.

## Requirements

- Python 3.x
- Tkinter (usually comes pre-installed with Python)
- Requests library (install with `pip install requests`)
